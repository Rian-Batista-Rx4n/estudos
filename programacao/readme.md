# 💻 Programação

Esta pasta reúne todas as minhas anotações, estudos e documentações relacionadas a **linguagens de programação**, **tecnologias de desenvolvimento** e **ferramentas utilizadas no dia a dia como dev**.

O objetivo aqui é criar um ambiente organizado onde cada linguagem possui sua própria estrutura de aprendizado, incluindo:

- Conceitos fundamentais  
- Sintaxe  
- Exemplos práticos  
- Exercícios  
- Projetos pessoais  
- Módulos avançados  

---

## 📂 Linguagens e Ferramentas Documentadas

Atualmente, o estudo está organizado nas seguintes pastas/languages:

- **Python3** — minha linguagem principal para backend e automações  
- **JavaScript** — base para web, navegadores, interfaces e lógica  
- **HTML5** — marcação estrutural para páginas web  
- **CSS3** — estilização e criação de layouts  
- **Bash** — scripts, automação e terminal Linux  
- **DAX** — linguagem de medidas e cálculos no Power BI  
- **M (Power Query)** — transformação de dados e ETL no Power BI  
- **MySQL** — banco de dados relacional, consultas e estrutura de dados

Cada linguagem possui seu próprio README e uma organização interna com aulas numeradas, seguindo uma trilha de aprendizado consistente.
